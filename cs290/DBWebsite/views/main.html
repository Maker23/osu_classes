How to use the Pinterest API

An interactive guide written for CS290, W17

Authenticating Users

Pinterest API calls need to be made on behalf of a Pinterest user, so the first thing you need to do is get your website user to establish their Pinterest credentials. Pinterest uses oAuth 2.0 for this.

Searching pins

Once the user is authenticated pins can be searched with a simple GET command. Only pin descriptions are searched with the default interface, but more complex searches can be done on the client side by retrieving all the user's pins and examining their fields.

Retrieving boards

Boards, like pins, can be accessed with a GET command. Boards only have a few bits of data associated with them - name, URL, and ID.
